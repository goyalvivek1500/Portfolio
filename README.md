A static personal portfolio webpage built as part of the **Frontend Development Internship — Batch 2026**.

## Tech Stack

- CSS3 (Grid + Flexbox, External only)

- CSS Grid for page layout, Flexbox for component alignment
- DRY CSS with reusable utility classes
- WCAG accessibility standards (alt text, aria-labels, focus styles)
- Responsive design (Tablet + Mobile breakpoints)
- Mobile hamburger navigation
